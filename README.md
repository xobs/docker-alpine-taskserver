Taskserver for Docker
=====================

Taskserver running under Docker, with compatible LetsEncrypt certs.

Starting a Server
-----------------

````
docker run \
    --rm \
    --detach \
    --name taskwarrior \
    -p 53589:53589 \
    -v /etc/letsencrypt:/certificates:ro \
    -e CN=[real domain name] \
    -v [local-server-data]:/data \
    xobs/taskwarrior
````

Then copy [local-server-data]/certs/client.* to your client.

You can create a new user by running:

````
# Create an organization called 'Public'
docker exec -it taskwarrior taskd add org Public
# Add a user, and take note of the output
docker exec -it taskwarrior taskd add user 'Public' 'First Last'
````

Copy client.cert.pem and client.key.pem from your server's data config directory.
Then, configure your copy of task warrior:

````
$ task config taskd.certificate -- [path-to]client.cert.pem
$ task config taskd.key         -- [path-to]client.key.pem
$ task config taskd.ca          -- [path-to]ca.cert.pem
$ task config taskd.server      -- host.domain:53589
$ task config taskd.credentials -- [org]/[First Last]/[User ID returned from 'taskd add user']
````

Finally, ensure you can sync:

````
$ task sync init
````

If you have TLS trouble, set debug.tls and try again:

````
$ task config debug.tls -- 2
````
